Penjelasan Aplikasi
- File yang pertama kali di jalankan adalah index.html
- Setelah login akan masuk ke 'process.php'. file tersebut berfungsi untuk mengecek jika user memasukkan nama atau tidak dan menyimpannya dalam session
- Lalu next masuk ke file 'math.php'. file ini adalah file utama untuk menjalankan loop program matematika
- setelah submit dari 'math.app' akan masuk ke file 'check.php' untuk mengecek apakah jawabannya benar atau salah. jika salah nyawa berkurang satu dan jika benar score bertambah 50
- Jika nyawanya sudah habis atau 0, maka akan dialihkan ke file 'over.php'. Didalamnya ialah tampilan akhir aplikasi dan Hall of Fame atau Score Board
